This is my ```.vimrc``` file.
   
Short, simple, effective.

```
" Enable Syntax Highlighting (TODO, FIXME, etc)
syntax on

" Show line numbers
set number

" Highlight search results
set hlsearch

" Show ruler (column numbers)
set ruler

" Highlight long lines
set colorcolumn=81,101

" Disable the bell
set visualbell

" Spaces & tabs
set tabstop=4
set softtabstop=4
set expandtab
set shiftwidth=4

" Highlight matching braces, parens, etc
set showmatch

" Automatically add closing characters
inoremap { {}<Esc>ha
inoremap [ []<Esc>ha
inoremap ( ()<Esc>ha
```
